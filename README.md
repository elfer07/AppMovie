# MovieApp
Application with a clean architecture this built with
 - MVVM
 - Extension Functions
 - Retrofit
 - Navigation Components
 - ViewBinding
 - Coroutines 

 # Support
 You need create a API key from https://www.themoviedb.org/documentation/api
